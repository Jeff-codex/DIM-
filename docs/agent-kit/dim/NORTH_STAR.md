# DIM — NORTH STAR

## 1. 프로젝트 정의

DIM은 스타트업, 서비스, 런칭, 산업 변화를
빠른 소식이 아니라 해석 가능한 피처로 남기는 한국어 매거진이다.

## 2. 프로젝트 본질

DIM의 핵심은 정보 전달 자체가 아니다.
핵심은 `해석 권위`를 축적해
다시 찾게 되는 에디토리얼 아카이브를 만드는 것이다.

## 2-1. DIM의 해석 철학: 본찰력

DIM은 `본찰력`이라는 고유의 비즈니스 분석 철학에 기반해 대상을 다시 읽는다.

본찰력은 사물·현상·정보의 겉면에 반응하지 않고, 그 이면에서 작동하는 구조·의도·힘의 흐름·본질적 메커니즘을 꿰뚫어 보는 능력이다.
즉, 무엇이 보이느냐보다 왜 그렇게 보이게 되었는가, 무엇이 말해졌느냐보다 무엇이 숨겨져 있는가, 지금 무엇이 일어났느냐보다 그 일을 가능하게 만든 근본 구조가 무엇인가를 읽어내는 힘이다.

DIM 문법으로 말하면 본찰력은 단순한 `깊이 보기`가 아니다.
정보를 많이 아는 능력도, 트렌드를 빨리 아는 능력도 아니다.

DIM의 본찰력은 아래의 힘으로 정리된다.

1. 사건을 사실로만 보지 않고 구조로 보는 힘
2. 표현을 메시지로만 보지 않고 의도로 보는 힘
3. 성과를 결과로만 보지 않고 시스템으로 보는 힘
4. 변화를 유행으로만 보지 않고 시대의 방향으로 읽는 힘

즉 본찰력은 `무엇이 화제인가`를 좇는 시선이 아니라, `무엇이 구조를 바꾸고 있는가`를 판별하는 시선이다.

DIM은 본찰력을 아래처럼 실무에 적용한다.

1. 무엇이 나왔는지보다 무엇이 바뀌는지를 먼저 쓴다
2. 기능 설명보다 구조 변화와 운영 맥락을 먼저 드러낸다
3. 판단문은 요약이 아니라 왜 지금 중요한지에 대한 편집 결론으로 남긴다
4. 브랜드 소개보다 시장·유통·업무 흐름 속 위치를 함께 설명한다
5. 원문을 옮기지 않고, 비교 지점과 맥락을 더해 읽을 이유를 만든다

## 3. 현재 공개 제품 정의

- 공개 라우트
  - `/`
  - `/articles`
  - `/articles/[slug]`
  - `/about`
  - `/submit`
- 운영 방식
  - 로컬 MDX 기반 피처 운영
  - 직접 큐레이션한 대표 이미지 사용
  - 프로젝트 제출을 받아 편집 및 해석 후 피처로 정리
- 기술 기반
  - Next.js 16
  - Cloudflare Workers
  - Cloudflare Pages preview

## 4. 핵심 사용자

- 구조를 읽고 싶은 창업가와 운영자
- 단순 보도보다 해석을 원하는 제품/전략 실무자
- 브랜드와 서비스의 방향을 깊게 보고 싶은 독자
- DIM에 피처 제안을 남기고 싶은 팀

## 5. 사용자가 기대하는 것

- 단순 소개보다 맥락과 판단
- 빠른 뉴스보다 다시 읽을 수 있는 피처
- 축적될수록 더 신뢰되는 아카이브
- DIM만의 시선이 남는 문장과 구조

## 6. 절대 목표

- DIM만의 해석 권위를 누적한다
- 브랜드 검색과 주제 검색에 모두 강한 피처 자산을 만든다
- 제출 기반 편집 구조를 신뢰 가능한 입구로 만든다
- 장기적으로 시리즈, 허브, 교육/포럼 확장 기반을 만든다

## 7. North Star Metric

1차
- 고품질 피처 누적 수

2차
- 검색 기반 재방문 비율

3차
- 브랜드명 직접 유입과 인용 증가

4차
- 피처 제안 유입과 재방문 독자 증가

## 8. 현재 단계 정의

현재 DIM은 `1차 공개 버전 + 매거진 문법 리디자인 완료` 단계다.

지금 필요한 것은:

- DIM 고유 카피와 카테고리 톤 정교화
- 기사 상세 리듬 개선
- 피처 제안 경험 정리
- 권위형 허브/시리즈로 이어질 다음 구조 설계

지금 불필요한 것은:

- 조급한 수익화
- 과한 CMS 확장
- 회원/댓글/결제 같은 부가 기능
- 뉴스룸형 속보 구조

## 9. 편집 판단 원칙

1. DIM은 정보량보다 해석 품질로 차별화한다
2. 제출된 정보는 그대로 싣지 않고 맥락을 더해 다시 정리한다
3. 검색 유입은 결과이고 권위 축적이 원인이다
4. 피처는 단발성 소개가 아니라 아카이브 자산이어야 한다
5. 장기적으로 교육, 포럼, 강연으로 이어질 수 있는 브랜드를 만든다
